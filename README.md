# 通讯录管理系统 - 前端

## 项目简介
基于Web的通讯录管理系统前端，采用原生HTML、CSS和JavaScript开发，实现联系人信息的增删改查功能。

## 技术栈
- HTML5
- CSS3
- 原生JavaScript
- LocalStorage (本地存储)

## 项目结构
832302208_contacts_frontend/
├── src/                           # 源代码目录
│   ├── contacts.html              # 主页面文件
│   ├── script.js                  # JavaScript逻辑文件
│   └── style.css                  # 样式文件
├── README.md                      # 项目说明文档
└── codestyle.md                   # 代码规范文档

## 功能特性
- 联系人信息管理（增删改查）
- 分组管理
- 搜索功能
- 生日提醒
- 响应式设计
- 本地数据持久化

## 快速开始
1. 克隆项目到本地
2. 直接在浏览器中打开 `contacts.html` 文件
3. 或使用本地服务器运行（推荐）

## 部署说明
项目为纯静态文件，可部署到任意Web服务器或CDN。

## 浏览器兼容性
支持现代浏览器（Chrome 60+、Firefox 55+、Safari 12+、Edge 79+）
